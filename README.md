# Matemáticas Aplicadas con Python

## Descripción

Este repositorio reúne los laboratorios, talleres y ejercicios desarrollados durante el curso de **Matemáticas Aplicadas** de la Universidad Icesi.

Los notebooks implementan diferentes algoritmos y procedimientos matemáticos utilizando Python para resolver problemas computacionales y analizar resultados.

---

## Contenido

El repositorio incluye actividades como:

- Laboratorios del curso.
- Talleres prácticos.
- Implementaciones de algoritmos en Python.
- Resolución de problemas matemáticos mediante programación.
- Simulaciones y ejercicios computacionales.

---

## Tecnologías utilizadas

- Python
- Jupyter Notebook
- NumPy
- Matplotlib

---

## Conceptos aplicados

- Programación científica
- Álgebra y cálculo computacional
- Visualización de datos
- Algoritmos matemáticos
- Resolución de problemas mediante programación

---

## Objetivo

Aplicar herramientas de programación para resolver problemas matemáticos, analizar resultados y fortalecer el razonamiento computacional.
